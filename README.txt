1. Judul Proyek
---------------
Fox Feast Restaurant Website - Website Restoran Bertema Rubah

2. Deskripsi Proyek
-------------------
Fox Feast Restaurant Website adalah platform online untuk restoran bertema rubah yang menyajikan pengalaman digital yang hangat dan menyenangkan. Website ini dirancang untuk:

- Memperkenalkan konsep unik restoran bertema rubah
- Menampilkan menu makanan dan minuman unggulan
- Memfasilitasi reservasi online
- Menampilkan galeri foto restoran dan hidangan
- Menyediakan informasi kontak dan lokasi
- Menampilkan testimoni pelanggan

Website ini sepenuhnya responsif dan bekerja optimal di semua perangkat, dari desktop hingga mobile.

3. Teknologi yang Digunakan
---------------------------
- HTML5            : Struktur dasar website
- Tailwind CSS     : Framework CSS untuk styling responsif
- JavaScript Murni : Untuk interaktivitas dan fungsionalitas
- Font Awesome     : Untuk ikon-ikon dekoratif
- Google Fonts     : Font Poppins untuk tipografi

-----------------------------------------

Tidak perlu instalasi atau setup lokal. Website dapat langsung diakses melalui link berikut:
